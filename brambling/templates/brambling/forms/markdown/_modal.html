Markdown on Dancerfly

Dancerfly uses a syntax known as "Markdown" in a few places. It's an easy-to-read, easy-to-write format that we convert into HTML for you!

Paragraphs

Paragraphs in Markdown are just one or more lines of consecutive text followed by one or more blank lines.

This is going to be a really awesome dance event!

Make sure you bring all your awesomeness from home too!

Headings

You can make headings by putting one to six # symbols in front of your heading text. More # symbols means smaller headings. Be careful with these!

# The largest heading
...
###### The smallest heading

Bold & Italic

Use * or _ to mark text as bold or italic.

*This text* and _this text_ will be italic.
**This text** and __this text__ will be bold.

Lists

You can make lists by putting a * or - at the beginning of the lines.

* Item
* Item
* Item

- Item
- Item
- Item

You can also make numbered lists by using numbers instead.

1. Item 1
2. Item 2
3. Item 3

Links

Links will automatically be converted, but you can also write links with custom text. All you have to do is wrap the text in brackets ([]) and then wrap the full link in parentheses (()).

For example, to create a link to dancerfly.com but with the link text "Check out Dancerfly!" you'd write this markdown:

[Check out Dancerfly!](https://www.dancerfly.com)
